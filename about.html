# 我是谁

H2Aoi，又称氢化葵

其实 H2Aoi 这个名字的正确写法是 H2Aoi，即 2 个 H+与 1 个 Aoi2-离子组合成的化合物。

不过上下角标这个实在是打不出来（Markdown 里当然可以），就变成了 H2Aoi，毕竟有一句话这么说来着

其实地上本没有路，走的人多了， 也便成了路 —— 鲁迅

一般平台的名字就叫这个，就酷安不一样。在酷安，我的用户名是泽渡雫

泽渡雫，即沢渡雫，美少女万華鏡 忘れな草と永遠の少女的女主角

当年（2017 年）玩完万华镜 2 后我就喜欢上了泽渡雫，所以酷安 id 就取了她的名字

不过不要以 id 取人！！！！
