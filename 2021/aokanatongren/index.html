记录一下苍彼同人游戏的开发过程（目前处于停更状态，高考后恢复）

目前想做的大概就下面几个一个

  • 《苍之彼方的四重奏》Asuka’s After Days

这个做好了以后还会有的，耶

# Asuka’s After Days

这玩意目前采用 TynaroBuilder Visual Novel Studio 制作，好处是可视化以及多平台兼容，坏处其实也很大

打算以后用 Unity 重造，不过是画大饼了

目前进度

# Demo1.0(2020.2.7)

一开始制作的版本，未添加任何剧情，纯属技术测试（主要是测试各项功能是否正常）

该版本内容

  • 一个像样的标题界面 + 主界面 BGM（与本篇主界面 BGM 一致）

BGM：

  • 一个像样的游戏界面（

  • 保存 / 读取进度等各项功能

啊这个 Demo 太粗糙了所以直接看下面的

# Demo1.1(2020.2.7)

做完 1.0 就忍不住做了 1.1

1.1 比 1.0 有了很大的提升，那就是有了剧情

剧情照搬原作广播剧 Dating Diaries 内容

由于是 Demo 演示版本，广播剧的语音没放进去（懒），正式版肯定是没有语音的（我上哪整哦）

我甚至往 b 站放了演示视频

该版本附加内容

  • 比较像样的剧情（搬运工）

  • BGM 平滑切换
  • Demo 结束提示

差不多就这样，现在就是缺个剧本，有意向可以在下面留言

我做苍彼同人游戏纯用爱发电，也就图个乐吧，可能会弃坑（大概率不会）
