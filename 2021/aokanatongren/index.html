# Asuka’s After Days

这玩意目前采用 TynaroBuilder Visual Novel Studio 制作，好处是可视化以及多平台兼容，坏处其实也很大

  • 保存 / 读取进度等各项功能

# Demo1.1(2020.2.7)

做完 1.0 就忍不住做了 1.1

1.1 比 1.0 有了很大的提升，那就是有了剧情

剧情照搬原作广播剧 Dating Diaries 内容

由于是 Demo 演示版本，广播剧的语音没放进去（懒），正式版肯定是没有语音的（我上哪整哦）

我甚至往 b 站放了演示视频

该版本附加内容

  • 比较像样的剧情（搬运工）

  • BGM 平滑切换
  • Demo 结束提示
