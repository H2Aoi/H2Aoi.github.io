# 前言

身为一名酷友，熟练的使用各种操作系统一定不是什么难事。windows,macos,linux 已经被很多酷友熟练的掌握，这也是一件好事。众所周知，windows 是一个适合玩游戏的平台，但是对于 linux 用户来说，这就是一件难事，尤其是对于 linux 环境下需要游玩 galgame 的用户来说，就更痛苦了。因此，我就写了这篇图文，不敢说肯定解决这种问题，可以说是一种探索吧

我觉得我说的这些大部分酷友都知道

# 此图文需要的环境

我使用的 linux 发行版为 Ubuntu 18.04.3，作为一个长期支持版本，我认为它会比 19.04 更加稳定，更适合我们进行这次的探索。对于其他 linux 发行版用户来说，我觉得可以举一反三，因此不作其他说明

图文环境为 virtualbox 虚拟机，因此可能会出现配置低的情况，请见谅

我又在实机上安装了 Deepin 20.1，应该比虚拟机性能要好一点，试了试 Steam Play

# 探索历程

# 使用 Virtualbox 虚拟机

galgame 作为扎根于 windows 的游戏，当然是要在 windows 环境下运行最适合啦。使用虚拟机的方法具体就不说了，都用 linux 了还会不知道这些吗

# 使用 Wine

wine 可以说是 linux 下运行 windows 程序的唯一办法了。基于大部分 galgame 都是 exe 格式，使用 wine 也是我们的一个必要选择了

wine 官网： https://appdb.winehq.org/

按照官网的教程安装好 wine 后，直接来到相应文件夹

这里选取了几个 Galgame：三色绘恋、茸雪、梦末、nekopara vol1。别问我为什么选这些，因为我只有这些

经过我的测试，这几个 Galgame 中基本没有能运行的，也许是因为这些游戏是 steam 游戏的原因吧。但是这种方法我并不推荐，因为 wine 本体兼容性实在是很低啊


运行失败QAQ

也许某些非 steam 游戏可以运行，反正其他方法都是基于 wine 的，为何不装呢

# 使用 Playonlinux

Playonlinux 是一款基于 wine 的程序，它比 wine 兼容性更好

Playonlinux 官网：playonlinux.com

安装完成后打开，是这个界面

选择安装一个程序，再点安装未在列表中的程序

选择在新虚拟盘中安装程序，输入一个英文名称，再选配置 wine

创建虚拟盘这一步，选择游戏支持的系统位数

再选择安装文件 exe

这样应该就能运行了，三色绘恋应该是可以的


来自贴吧

每一次进入游戏都要重新设置一遍，存档都会在的

根据网友说法，这两款游戏也是能运行的

# 使用 Steam

对没错，就是那个 G 胖的平台 Steam。steam 平台为 linux 用户专门开发了 steam play 这个程序，现在可以说是对 linux 下游戏最好的一个选择了

安装完 steam 后打开进入到设置，勾选 “Enable Steam Play for supported titles"和" 为所有其他产品启用 Steam Play"

这时你点进自己的游戏，会发现右边会多出一些东西，这就说明你对该游戏启用了 Steam Play

选择左下角的添加游戏，把自己的游戏添加进去。再在游戏属性里面勾上最后一个

点击开始就可以打开了（不敢说一定打得开，我也打不开）

由于我的下载速度实在是太慢，只好拿出～ 划时代大作 Answer The Question 了

可以看到，游戏是可以运行的

我下载了 LoveChoice 这款游戏，由于缺少 dx11 所以无法运行。不过这游戏有 Linux 版本所以就能运行了。不过我以前试过，相当一部分游戏是可以打开的

相同的，有很多游戏都可以通过这种方法运行，具体我就不多说了（没下游戏）

既然虚拟机啥也跑不起来那为什么不试试实机呢？

把我库里的 galgame 全试了个遍，目前发现这么这么几个可以跑的

  • Tricolour Lovestory

对没错，我库里就这玩意跑得起来，其他都是出现画面后立刻闪退

# 游玩移植版

这个没试过，应该比用 wine 运行效率高一点。可以去 Linux 游戏吧找找

# 总结

经过我的探索，我得出了一个结论：Linux 对于 Galgame 玩家实在是很不友好。我的建议就是干脆装个双系统，虚拟机的效率也没有实体机高。不过既然都用 Linux 了，一点小折腾算什么呢？不说了，我还是用我的 windows10 玩 Galgame 吧

本文其实是属于探索性质的一篇文章，并没有多少技术含量，大佬误喷我，我又不是一个 linux 用户

为了自己心爱的女孩，多折腾一下算什么呢？
