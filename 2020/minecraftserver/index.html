如何运用云主机开Minecraft官方BDS服务器

2020年12月18日 515 字 大概 2 分钟
  1. 1. 云主机的选购
  2. 2. ssh的配置
  3. 3. 开服！

本文还是第一次发于酷安，不过做了点修改，酷安那边就不改了

云主机的选购

我自己开服用的是阿里云的学生机，一个月才10块钱，还是可以的，虽然有点卡。。。。不过有钱的话，阿里云腾讯云都行的

百度云翼计划开发者成长计划或者传送门，点第一个

这个只要低于24岁就可以买，还是比较好的

选系统镜像，系统选Ubuntu18.04（最新版本即可）或者Centos（需要自行解决依赖）

购买之后进入ssh

ssh的配置

手机可以选择juicessh，windows可以选putty，linux就用终端吧

这里用juicessh做示范

点击连接，然后变成这样

地址填云服务器的地址，认证就新建一个认证，用户名填root，密码填设定的密码

然后就可以连接啦！记得开放22端口，虽然默认是开着的

开服！

连好ssh，大概是这样的

这边建议先去添加一些软件源，方法自行百度，这里就不写出来了

接下来输入指令

sudo apt-get update
sudo apt-get install libcurl3 build-essential

这个玩意需不需要其实我也不太清楚，不过防范于未然嘛

然后接着输入

wget https://minecraft.azureedge.net/bin-linux/bedrock-server-1.16.201.02.zip

下不动的话可以用scp指令传上去，具体自己百度。这个链接会有变化，可以到[这里](Minecraft Server Download | Minecraft)获取（选Ubuntu那个）

解压后进入目录，输入

LD_LIBRARY_PATH=. ./bedrock_server

大概这样就可以了，我开了很久所以是这样的

然后。。。就好了。。。才怪

进入云服务器控制台安全组

添加图里最上面的两条规则（也许只用UDP）

然后是真的好了，也需要重启一下

如果想要在退出ssh的时候保持运行，就用screen吧！

用如下指令

sudo apt-get install screen
screen

挂起就按Ctrl+A+D

这样退出ssh后也可运行
