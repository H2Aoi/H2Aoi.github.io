# Galgame 的历史

根据萌娘百科，Galgame 一词首先是用来称呼日本通信网络（日本テレネット）的旗下游戏《梦幻战士》（「梦幻戦士ヴァリス」），因这款游戏的内容等含有 Galgame 的要素，于是被认为是初代 Galgame

但萌娘百科也指出，Galgame 一词最早出现于 1992 年 3 月 27 日的 FAMI 通，但在此之前已经存在 Galgame 了

百度百科指出，早在 1982 年，日本光荣公司推出了一款叫 Night Life 的美少女游戏，这款游戏与今天的 Galgame 相比起来有很大的不同，却是有记载的最早的一个此类作品，故我们可以认为这款游戏是初代 Galgame

# 关于 Galgame 的一些误区

# Galgame 一定有涩青内容

这可能是最大的一个误区了，事实上，很多优秀的 Galgame 并没有涩青内容的设定（不包括某些同人作品），比如说。。。。我也忘记了

# Galgame 一定需要用电脑进行游玩

随着技术的进步，在 Android 设备（我不知道 ios 上有没有）上已经出现了很多 Galgame 模拟器，krkr 就是其中一个比较好的模拟器

# Galgame 只适合男生游玩

不不不，女生当然也是可以玩的，谁管你呢

# Galgame 是一种视觉小说，没有分支选项

事实上，很多 Galgame 为了丰富剧情，会添加一些分支结局及选项（比如君与彼女与彼女之恋）

# 我该如何去选择 Galgame

# 我还是个新人呜呜呜

如果你还是一个刚刚接触 Galgame 的新人，那么我推荐去游玩《CLANNAD》

这是由是由日本游戏品牌 Key 所发行的一款恋爱冒险游戏，并有许多衍生作品

《CLANNAD》的动画版也被许多人誉为是神作，可见其剧情的丰富（虽然我没看过）

作为一个新人，你可以先通过这款游戏了解 Galgame 的主要结构及内容，再通过 Key 社的其他作品，进入下一阶段

# 就这？我已经接触过一些 Galgame 了

如果你已经接触过一些 Galgame，我推荐去游玩《美少女万花镜》系列 2-4（不包括 2.5）

我的头像及名称就来自于这个系列的第二部勿忘草与永远的少女中的泽渡雫，可见我对这系列游戏的喜爱程度（当然是指酷安头像和名称辣）

这是由是由 ωstar 制作的系列超自然学风格官能 AVG 游戏，画面精美，适合 lsp 对内容有一定要求的人，并且 bgm 都比较耐听，可谓是买音乐送游戏了

当然只玩这几部当然不够，还可以接触一些优秀作品，比如说千恋万花苍之彼方的四重奏！

# 总结

游玩 Galgame 其实是一种很好的消磨时间的方式，基本打通一部 Galgame 需要一个小时以上的时间，有些还能让你学到很多，我觉得还是不错的

有些人对于 Galgame 可能存在偏见，认为玩家是在逃避现实。但是我觉得，我们并不是在逃避现实，现实对我们太残酷了，我们需要一个精神的避风港，去更好的面对现实，Galgame 就是一个很好的避风港
